# WikiHow 知识库系统 - 需求规格说明书

## 1. 项目概述

### 1.1 项目背景
仿照 WikiHow 网站，实现一个基于 Spring Boot + Vue 的知识库系统，支持步骤式图文教程的创作与展示。

### 1.2 项目目标
- 用户可以注册登录、发布和浏览教程文章
- 文章支持步骤式展示（WikiHow 核心特色）
- 基础的互动功能（评论、点赞、收藏）

### 1.3 技术栈
| 层级 | 技术 |
|------|------|
| 前端 | Vue 3 + Vite + Pinia + Vue Router + Element Plus |
| 后端 | Spring Boot 3 + MyBatis-Plus + Java 17 |
| 数据库 | MySQL 8.0 |

---

## 2. 用户角色

### 2.1 游客
- 浏览文章列表和详情
- 搜索文章
- 查看分类

### 2.2 注册用户
- 登录/注册/修改个人资料
- 创建、编辑、删除自己的文章
- 评论、点赞、收藏文章

### 2.3 管理员
- 管理所有文章（删除、下架）
- 管理用户（禁用）
- 管理分类

---

## 3. 功能需求

### 3.1 用户模块

| 功能 | 说明 |
|------|------|
| 注册 | 用户名 + 邮箱 + 密码 |
| 登录 | 用户名/邮箱 + 密码，返回 Token |
| 个人资料 | 修改头像、昵称、简介 |
| 修改密码 | 验证旧密码后修改 |

### 3.2 文章模块

**文章结构：**
```
文章
├── 标题、摘要、封面图
├── 分类、标签
├── 难度（简单/中等/困难）
└── 步骤列表
    ├── 步骤标题
    ├── 步骤内容
    └── 步骤图片
```

| 功能 | 说明 |
|------|------|
| 文章列表 | 分页、按分类筛选、按时间排序 |
| 文章详情 | 步骤式展示、作者信息、互动数据 |
| 创建文章 | 富文本编辑、步骤编辑、图片上传 |
| 编辑文章 | 仅作者可编辑 |
| 删除文章 | 仅作者或管理员可删除 |

### 3.3 分类模块

- 二级分类结构（如：技术教程 > 编程开发）
- 管理员可增删改分类

### 3.4 搜索模块

- 按标题关键词搜索
- 按分类筛选
- 按时间排序

### 3.5 互动模块

| 功能 | 说明 |
|------|------|
| 评论 | 发表评论、删除自己的评论 |
| 点赞 | 文章点赞/取消 |
| 收藏 | 文章收藏/取消、查看收藏列表 |

### 3.6 后台管理

| 功能 | 说明 |
|------|------|
| 用户管理 | 查看用户列表、禁用用户 |
| 文章管理 | 查看所有文章、删除文章 |
| 分类管理 | 增删改分类 |

## 5. 接口清单

### 认证接口
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/auth/register | 注册 |
| POST | /api/auth/login | 登录 |

### 用户接口
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/users/:id | 获取用户信息 |
| PUT | /api/users/:id | 更新用户信息 |

### 文章接口
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/articles | 文章列表 |
| GET | /api/articles/:id | 文章详情 |
| POST | /api/articles | 创建文章 |
| PUT | /api/articles/:id | 更新文章 |
| DELETE | /api/articles/:id | 删除文章 |

### 分类接口
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/categories | 分类列表 |

### 互动接口
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/comments | 发表评论 |
| DELETE | /api/comments/:id | 删除评论 |
| POST | /api/likes | 点赞/取消 |
| POST | /api/favorites | 收藏/取消 |
