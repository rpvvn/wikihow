## 3. 数据表设计

### 3.1 用户表 (user)

| 字段       | 类型         | 非空 | 说明                      |
| ---------- | ------------ | ---- | ------------------------- |
| id         | BIGINT       | 是   | 主键                      |
| username   | VARCHAR(50)  | 是   | 用户名，唯一              |
| email      | VARCHAR(100) | 是   | 邮箱，唯一                |
| password   | VARCHAR(255) | 是   | 密码(BCrypt加密)          |
| nickname   | VARCHAR(50)  | 否   | 昵称                      |
| avatar     | VARCHAR(255) | 否   | 头像URL                   |
| bio        | VARCHAR(500) | 否   | 个人简介                  |
| role       | VARCHAR(20)  | 是   | 角色: USER/REVIEWER/ADMIN |
| status     | TINYINT      | 是   | 状态: 0禁用/1正常         |
| created_at | DATETIME     | 是   | 创建时间                  |
| updated_at | DATETIME     | 是   | 更新时间                  |

```sql
CREATE TABLE `user` (
  `id` BIGINT NOT NULL AUTO_INCREMENT,
  `username` VARCHAR(50) NOT NULL,
  `email` VARCHAR(100) NOT NULL,
  `password` VARCHAR(255) NOT NULL,
  `nickname` VARCHAR(50) DEFAULT NULL,
  `avatar` VARCHAR(255) DEFAULT NULL,
  `bio` VARCHAR(500) DEFAULT NULL,
  `role` VARCHAR(20) NOT NULL DEFAULT 'USER',
  `status` TINYINT NOT NULL DEFAULT 1,
  `created_at` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
  `updated_at` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  PRIMARY KEY (`id`),
  UNIQUE KEY `uk_username` (`username`),
  UNIQUE KEY `uk_email` (`email`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

### 3.2 分类表 (category)

| 字段        | 类型         | 非空 | 说明       |
| ----------- | ------------ | ---- | ---------- |
| id          | BIGINT       | 是   | 主键       |
| name        | VARCHAR(50)  | 是   | 分类名称   |
| description | VARCHAR(500) | 否   | 分类描述   |
| cover_image | VARCHAR(255) | 否   | 分类封面图 |
| parent_id   | BIGINT       | 否   | 父分类ID   |
| sort_order  | INT          | 是   | 排序       |
| created_at  | DATETIME     | 是   | 创建时间   |

```sql
CREATE TABLE `category` (
  `id` BIGINT NOT NULL AUTO_INCREMENT,
  `name` VARCHAR(50) NOT NULL,
  `description` VARCHAR(500) DEFAULT NULL,
  `cover_image` VARCHAR(255) DEFAULT NULL,
  `parent_id` BIGINT DEFAULT NULL,
  `sort_order` INT NOT NULL DEFAULT 0,
  `created_at` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
  PRIMARY KEY (`id`),
  KEY `idx_parent` (`parent_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

### 3.3 文章表 (article)

| 字段            | 类型         | 非空 | 说明                                        |
| --------------- | ------------ | ---- | ------------------------------------------- |
| id              | BIGINT       | 是   | 主键                                        |
| title           | VARCHAR(200) | 是   | 标题                                        |
| summary         | VARCHAR(500) | 否   | 摘要                                        |
| cover_image     | VARCHAR(255) | 否   | 封面图                                      |
| user_id         | BIGINT       | 是   | 作者ID                                      |
| category_id     | BIGINT       | 是   | 分类ID                                      |
| tags            | VARCHAR(255) | 否   | 标签(逗号分隔)                              |
| difficulty      | VARCHAR(20)  | 是   | 难度: EASY/MEDIUM/HARD                      |
| view_count      | INT          | 是   | 浏览数                                      |
| like_count      | INT          | 是   | 点赞数                                      |
| favorite_count  | INT          | 是   | 收藏数                                      |
| comment_count   | INT          | 是   | 评论数                                      |
| status          | TINYINT      | 是   | 状态: 0草稿/1待审核/2已发布/3已拒绝/4已下架 |
| is_outdated     | TINYINT      | 是   | 是否过时: 0否/1是                           |
| outdated_reason | VARCHAR(500) | 否   | 过时原因                                    |
| references      | TEXT         | 否   | 参考文献(JSON格式)                          |
| created_at      | DATETIME     | 是   | 创建时间                                    |
| updated_at      | DATETIME     | 是   | 更新时间                                    |

```sql
CREATE TABLE `article` (
  `id` BIGINT NOT NULL AUTO_INCREMENT,
  `title` VARCHAR(200) NOT NULL,
  `summary` VARCHAR(500) DEFAULT NULL,
  `cover_image` VARCHAR(255) DEFAULT NULL,
  `user_id` BIGINT NOT NULL,
  `category_id` BIGINT NOT NULL,
  `tags` VARCHAR(255) DEFAULT NULL,
  `difficulty` VARCHAR(20) NOT NULL DEFAULT 'MEDIUM',
  `view_count` INT NOT NULL DEFAULT 0,
  `like_count` INT NOT NULL DEFAULT 0,
  `favorite_count` INT NOT NULL DEFAULT 0,
  `comment_count` INT NOT NULL DEFAULT 0,
  `status` TINYINT NOT NULL DEFAULT 0,
  `is_outdated` TINYINT NOT NULL DEFAULT 0,
  `outdated_reason` VARCHAR(500) DEFAULT NULL,
  `references` TEXT DEFAULT NULL,
  `created_at` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
  `updated_at` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  PRIMARY KEY (`id`),
  KEY `idx_user` (`user_id`),
  KEY `idx_category` (`category_id`),
  KEY `idx_created` (`created_at`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

### 3.4 文章编辑锁表 (article_edit_lock)

| 字段       | 类型     | 非空 | 说明         |
| ---------- | -------- | ---- | ------------ |
| id         | BIGINT   | 是   | 主键         |
| article_id | BIGINT   | 是   | 文章ID，唯一 |
| user_id    | BIGINT   | 是   | 锁定用户ID   |
| locked_at  | DATETIME | 是   | 锁定时间     |
| expires_at | DATETIME | 是   | 过期时间     |

```sql
CREATE TABLE `article_edit_lock` (
  `id` BIGINT NOT NULL AUTO_INCREMENT,
  `article_id` BIGINT NOT NULL,
  `user_id` BIGINT NOT NULL,
  `locked_at` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,

  KEY `idx_user` (`user_id`),
  KEY `idx_expires` (`expires_at`)

```

## 6. 主要特性说明

### 6.1 文章版本控制
- 每次编辑文章时，创建新的版本记录
- `article_version` 表存储完整的内容快照
- 支持版本回滚和历史查看

### 6.2 文章编辑锁
- 防止多人同时编辑同一篇文章
- 锁定时间有过期机制，防止死锁
- 每篇文章最多只能有一个活跃锁

### 6.3 文章审核流程
- 文章发布前需要审核员审核
- 审核员可以通过或拒绝文章
- 记录审核意见和审核时间

### 6.4 过时内容管理
- 用户可以举报过时的文章
- 管理员可以标记文章为过时
- 支持记录过时原因和处理备注

### 6.5 系统通知
- 支持多种通知类型
- 关联文章和评论，便于用户快速定位
- 支持已读/未读状态追踪
