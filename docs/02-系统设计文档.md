## 4. 统一响应格式

```json
{
  "code": 200,
  "message": "success",
  "data": { }
}
```

| 状态码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 参数错误 |
| 401 | 未登录 |
| 403 | 无权限 |
| 404 | 不存在 |
| 500 | 服务器错误 |

## 6. 认证设计

使用 JWT Token 认证：

1. 用户登录成功后，返回 Token
2. 前端将 Token 存储在 localStorage
3. 请求时在 Header 中携带：`Authorization: Bearer {token}`
4. 后端拦截器验证 Token 有效性

```java
// JWT 工具类核心方法
public class JwtUtil {
    public static String generateToken(Long userId, String username) { }
    public static Claims parseToken(String token) { }
    public static boolean isExpired(String token) { }
}
```

## 7. 文件上传

- 上传接口：`POST /api/files/upload`
- 存储位置：`/uploads/` 目录
- 返回格式：`{ "url": "/uploads/xxx.jpg" }`
- 限制：图片类型，最大 5MB
